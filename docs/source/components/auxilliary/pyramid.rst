force-pyramid
=============

Usage
^^^^^

.. code-block:: bash

  force-pyramid

    Usage: force-pyramid [-hjrl] image [image]*

    -h  = show his help
    -j  = number of jobs
            defaults to 'as many as possible'
    -r  = resampling option
            default: nearest
    -l  = levels, comma-separated
            default: 2 4 8 16
